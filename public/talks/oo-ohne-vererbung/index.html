OO ohne Vererbung

Tag 2, 15:15 - 16:15 – Deutsch

In so gut wie jeder Definition von Objektorientierung findet sich ein Wort: Vererbung! In diesem Vortrag möchte ich auf die Probleme von Vererbung eingehen mit einem starken Fokus auf die Testbarkeit von Klassen. Danach werde ich Lösungen zeigen, wie man Vererbung umgehen kann und Fälle, in denen man wohl nicht um sie herumkommt.
